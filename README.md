The repository is organized by difficulty and project type to help track my progress:

* `01 - Beginner/`: Foundational circuits (Clippers, Logic Gates, Dividers).

* `docs/`: Design calculations, datasheets, and reference materials.
* `libs/`: Custom component libraries and symbol files.
