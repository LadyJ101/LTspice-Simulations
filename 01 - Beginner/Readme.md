# 01 - Beginner: Foundations of Electronics

## Overview
Welcome to the foundational module of my electronics simulation repository. This folder contains my initial experiments in LTspice, focused on understanding the building blocks of modern electronic circuits—ranging from individual transistor physics to basic digital logic.

The projects here follow a deliberate learning path: **Transistor Physics → Analog Circuits → Digital Logic.**

## Projects Included

| Project Folder | Description | Focus |
| :--- | :--- | :--- |
| `NMOS_Xtics` | Characterization of NMOS transistors | Physics & DC Analysis |
| `PMOS_Xtics` | Characterization of PMOS transistors | Physics & DC Analysis |
| `Clipper_Circuit` | Diode clipping experiments | Analog Signal Processing |
| `CMOS_NAND_GATE` | CMOS Logic implementation | Digital Design |
| `Half-Adder` | 1-bit arithmetic circuit | Logic Systems |

## How to use this folder
1.  **Start with the Transistors (`NMOS`/`PMOS`):** Understanding how these behave is key to everything else in this folder.
2.  **Move to Analog (`Clipper`):** Get comfortable with transient analysis and voltage probing.
3.  **Advance to Digital (`NAND` / `Half-Adder`):** Apply what you learned about transistors to create functional digital systems.

## Resources
* **Software:** LTspice
* **Documentation style:** Each project folder contains its own `README.md` and waveform screenshots for quick verification.
